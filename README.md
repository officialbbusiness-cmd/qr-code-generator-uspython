# QR Code Generator (Python)

## Description
This project is a simple Python script that generates a QR code from a user-provided URL and saves it as an image.

## Features
- Accepts URL input from the user
- Generates QR codes using the qrcode library
- Saves the QR code as a PNG file

## How to Run
1. Install dependencies:
   pip install qrcode[pil]

2. Run the script:
   python your_script_name.py

3. Enter a URL when prompted

## Example
Enter the url: https://google.com

## Output
A QR code image will be saved to your computer.
